# 🚀 Enterprise AI Document Agent

An automated AI solution built to process, analyze, and summarize corporate documents (PDF & TXT) using high-performance Open Source LLMs.

## ✨ Key Features
- **Multi-Format Support:** Processes both raw text and complex PDF structures.
- **Enterprise Brain:** Powered by Llama 3.3 via Groq for high-speed analysis.
- **Action-Oriented:** Allows users to download structured analysis reports instantly.
- **Secure Architecture:** Implements industry-standard `.env` security for API management.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **UI Framework:** Streamlit
- **AI Model:** Llama 3.3-70b (Meta)
- **Engine:** Groq Cloud API
- **PDF Engine:** PyPDF

## 🚀 How to Run Locally
1. Clone the repo.
2. Create a `.env` file and add your `GROQ_API_KEY`.
3. Run `pip install -r requirements.txt`.
4. Launch with `streamlit run app.py`.
